# P-Script_Get-EventLog

Script qui va chercher l'event log d'un parc informatique⠀⠀⠀⠀⠀⠀⠀⠀⠀⠀

--- 

## Exemple d'utilisation :

Voici un exemple pour utiliser le script :
```powershell
./Get-EventLogs.ps1 "C:/Users/User/Documents/listPCs.txt"
```
